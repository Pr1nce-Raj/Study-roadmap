# 📚 Study Roadmap Tracker

A personal study progress tracker built with vanilla HTML, CSS, and JavaScript. Track your learning roadmap, mark topics as complete, and sync progress across devices — all without any backend or login required.

🌐 **Live App:** [studytracker-ccb52.web.app](https://studytracker-ccb52.web.app) · [studytracker-ccb52.firebaseapp.com](https://studytracker-ccb52.firebaseapp.com)

---

## ✨ Features

- ✅ Mark topics and subtopics as complete
- 💾 Progress saved automatically via `localStorage`
- 🔄 File sync behaviour for consistent state
- 📱 Works on mobile and desktop browsers
- ⚡ No login, no backend, no dependencies — just open and use

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript (vanilla)
- **Hosting:** Firebase Hosting
- **Storage:** Browser `localStorage`

---

## 📁 Project Structure

- `index.html` — main app
- `favicon.png` — favicon
- `firebase.json` — Firebase hosting config
- `.firebaserc` — Firebase project settings
- `archive/` — older local versions

---

## 🚀 Run Locally

Just open `index.html` in any browser — no installation needed.

You can also use the **Live Server** extension in VS Code for a smoother development experience.
